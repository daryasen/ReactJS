# Инструкция по запуску приложения

## Быстрый старт для разработки

1. **Запустите сервер** (в первом терминале):
```bash
cd shop-app
npm run server
```

2. **Запустите клиент** (во втором терминале):
```bash
cd shop-app
npm run dev
```

3. Откройте браузер: http://localhost:3000

## Запуск production версии

1. **Соберите и запустите**:
```bash
cd shop-app
npm start
```

2. Откройте браузер: http://localhost:3001

## Данные для входа

- **Имя пользователя:** admin
- **Пароль:** admin

## Структура проекта

- `src/` - исходный код React-приложения
- `server/` - серверная часть (Node.js/Express)
- `dist/` - собранное приложение (создается после `npm run build`)

## API Endpoints

- `POST /api/auth/login` - авторизация
- `GET /api/goods?page=1&limit=10` - список товаров (требует токен)
- `GET /api/goods/:id` - товар по ID (требует токен)
